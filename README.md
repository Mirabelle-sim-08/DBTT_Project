
  # WelderCert Prototype

  This is a code bundle for User dashboard prototype. The original project is available at
  https://www.figma.com/design/1n7wrgrZWCx2Av4dkephBA/User-dashboard-prototype.

  ## About
  WelderCert is a cloud-based digital certification platform designed for aircraft Maintenance, Repair and Overhaul (MRO)
  organisations. It enables secure, efficient credential management and verification, allowing MRO personnel to scan QR codes on
  digital certificates to instantly verify technician qualifications, retrieve certification details, and manage 
  compliance documentation from any device.

  ## Key Screens
  - Login
  - Dashboard
  - Certification Verification
  - Audit Logs

  ## Tech Stack
  - React 18
  - Vite
  - html5-qrcode
  - Supabase

  ## Getting Started

  ### Prerequisites
  - Node.js
  - npm

  ### Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
