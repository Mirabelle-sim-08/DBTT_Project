---

**Prompt for Figma Make**

Create a **mobile app prototype for aviation welders to manage and verify digital certifications**.

### Scenario

The **welder is already logged in**. The app serves as a **Digital Certification Wallet** that allows welders to store, share, and verify their welding certifications with employers.

Design a **clean industrial-style mobile UI** suitable for technical workers (simple icons, strong typography, clear buttons).

---

# Screen 1: Home Dashboard

Purpose: Central hub for welders.

Components:

* Header: “Digital Welding Certification Wallet”
* Welder Digital ID card (Name, photo, certification ID)
* Button: **View Certifications**
* Button: **Share Certification**
* Button: **Download Certificate**
* QR Code preview for sharing credentials

Design elements:

* Card-based UI
* Industrial blue / grey color palette
* Simple icons (certificate, QR code, shield verification)

---

# Screen 2: Certification Wallet

Purpose: Store all welding certifications.

Components:

* List of certifications
* Each card shows:

  * Certification name
  * Issuing authority (e.g. CAAS / ICAO / EASA)
  * Expiry date
  * Status indicator (Valid / Expiring / Expired)
* Tap card to view full certification

Features:

* Download certificate
* Generate QR code
* Share with employer

---

# Screen 3: Certification QR Code

Purpose: Share certification with employers.

Components:

* Large QR code
* Certification summary
* Buttons:

  * Share
  * Download
  * Copy verification link

---

# Screen 4: Employer Verification Scanner

Purpose: Employers verify certifications.

Components:

* QR code scanner interface
* Camera view
* Button: “Scan Certification”
* Result screen showing:

  * Welder Name
  * Certification Type
  * Issuer
  * Expiry Date
  * Authenticity status (Verified / Invalid)

Security feature:

* Verified badge
* Prevents fake certificates

---

# Interaction Flow

Welder logs in → Views digital wallet → Generates QR code → Employer scans → Certification instantly verified.
